# База знаний методики «Теремок»

Сюда можно загрузить материалы, чтобы бот мог лучше понимать контекст и логику игры.

**Рекомендуемые форматы:**
- `.txt` (обычный текст)
- `.md` (Markdown)

**Что сюда положить:**
1. Транскрипцию вебинара.
2. Текст из презентаций.
3. Раздаточные материалы.
4. Описания типажей своими словами или из методички.
